## Stack
| **Databases** 	| **Server** 	| **Front** 	|
|---------------	|------------	|-----------	|
| SQL Server    	| .NET 8.0   	| Blazor    	|
| MongoDB       	| SignalR    	| MudBlazor 	|
| Redis         	| gRPC       	|           	|

| **Data access** 	| **Patterns**       	| **Messaging** 	|
|-----------------	|--------------------	|---------------	|
| EF Core         	| Clean Architecture 	| RabbitMQ      	|
|                 	| CQRS MediaR        	|               	|

**Docker** & **Kubernetes** support :ship: :rocket:
